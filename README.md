# React-Login-Status-App
## Overview
This project is a beginner-friendly React application that demonstrates how to manage component state using the useState hook. It shows a login and logout message based on user interaction and updates the UI dynamically.

## Features
- Uses React functional components
- Demonstrates useState hook
- Conditional rendering based on login status
- Login and Logout toggle button
- Simple and clean UI

## Technologies Used
- React JS
- JavaScript (ES6)
- Inline CSS styling

## Output
The application displays:
- "You are logged in" message when the user is logged in
- A button that toggles between Login and Logout states

## Project Structure
- App.js – Main component
- LoginStatus.js – Handles login/logout logic using state
- index.js – Application entry point


## Learning Outcome
This project helps beginners understand:
- React state management
- Conditional rendering
- Component-based architecture
- Handling user events in React

## Conclusion
The React Login Status Toggle App is a simple yet effective project to understand core React concepts. It is ideal for beginners and can be extended further by adding authentication, forms, or routing.
